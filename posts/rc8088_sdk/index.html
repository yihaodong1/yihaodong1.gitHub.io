RC8088 雷达芯片驱动 SDK

项目简介

本项目是 RC8088 毫米波雷达芯片 的嵌入式 Linux 驱动 SDK，基于 RK3588 平台 (aarch64) 开发。通过 SPI 总线与芯片通信，完成芯片初始化、寄存器配置、校准、波形控制及功能性安全监控等功能。

软件架构

┌──────────────────────────────────────────────────────────────┐
│        应用层 (main.c)                                     │  CFG 层 (RC8088_CFG)          ← 配置数据结构
├──────────────────────────────────────────────────────────────┤
│  FUNC 层 (2_RC8088_FUNC)      ← 功能 API 入口
├─────────────────────────────┤
│  HAL 层 (1_RC8088_HAL)        ← 硬件抽象
├─────────────────────────────┤
│  IF 层 (RC8088_IF)            ← SPI / GPIO / IRQ / Timer
├─────────────────────────────┤
  • IF 层：直接操作硬件接口（SPI 读写、GPIO 控制、中断、定时器），不依赖上层
  • HAL 层：封装硬件操作，提供寄存器读写、校准、控制等基础功能
  • FUNC 层：面向应用的功能 API，共 51 个函数，分为初始化、校准、配置下载、控制、功能安全 5 大类
  • CFG 层：芯片配置数据结构与默认值

功能特性

  • 双芯片 (chip1 & chip2) 同步控制
  • 寄存器读写验证
  • Profile 配置下载 & 动态重载
  • Subframe 波形切换
  • AFE 校准
  • MIPI 接口使能/禁用
  • ESR（功能安全）状态监控
  • 中断屏蔽与清除
  • Profile LUT 动态修改

编译构建

宿主机本地编译

1
2
3

mkdir -p build && cd build
cmake ..
make

交叉编译（RK3588 aarch64）

1
2
3

mkdir -p build && cd build
cmake -DCMAKE_TOOLCHAIN_FILE=../aarch64-toolchain.cmake ..
make

产物为 build/rc8088_app。

编译器要求

  • GCC 10.3+（交叉编译使用 gcc-arm-10.3-2021.07-x86_64-aarch64-none-linux-gnu）

运行方式

1
2

# 在目标板 (RK3588) 上
./rc8088_app

默认会执行：

  1. 寄存器读写通路验证 (testrdwr)
  2. 双芯片初始化 (RC8088_userInit)
  3. 进入主循环 (RC8088_userCfgLoop)：周期性 RF 启停 + ESR 监控

测试开关

在 main.c 顶部可启用/禁用以下测试：

宏定义 功能
RC8088_SINGLE_FREQ_TEST 单频点测试
RC8088_WAVE_SWITCH_TEST Subframe 波形切换测试
RC8088_PROFCFG_RELOAD_TEST Profile 配置动态重载测试
RC8088_PROFLUT_RELOAD_TEST Profile LUT 动态修改测试
