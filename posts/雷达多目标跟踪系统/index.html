雷达多目标跟踪系统 (AMEKFASSO20251218)

概述

本系统是一个完整的 FMCW/脉冲雷达信号处理与多目标跟踪 程序，基于 MATLAB R2022a 开发。系统从原始回波 .dat 文件读取双通道 IQ 数据，经过脉冲压缩、距离-多普勒成像（RDMAP）、CFAR 检测、双通道测角，最终通过 扩展卡尔曼滤波（EKF） 实现对多个运动目标的实时跟踪与可视化。

核心模块详解

1. 参数读取 (第6-52行)

从 .dat 文件头读取雷达工作参数：

参数 说明 文件偏移
PRF 脉冲重复频率 (Hz) 偏移118
AuxTp (Tr) 发射脉宽 (s) 偏移122
AuxBw (Bw) 信号带宽 (Hz) 偏移124
Auxfc (fc) 载波频率 (Hz) 偏移126
SigT_start 信号起始时间 偏移128
Fs 采样率 (Hz) 偏移138
LookDir 雷达照射方向 偏移162
birdmode 鸟视图模式 偏移182

2. 脉冲压缩 (第107-133行)

调用外部函数 compress_base20251209.m，对每个通道的每个脉冲进行匹配滤波：

src(t) = exp(j·π·K·t²) · Hamming窗    （参考信号）
data_com = conv(data, conj(src))        （匹配滤波）
  • bw1 = Kr·Bw/Fs：归一化带宽
  • pts2 = Tr·Fs：脉宽对应的采样点数
  • Hamming 窗用于抑制距离旁瓣

3. RDMAP 生成 (第282-390行)

内部函数调用链：

generateRDMAPWith3DDisplay()
  ├── generateDopplerVelocityAxis()    → 多普勒/速度轴
  └── generateRangeDopplerMap()        → 各通道RDMAP
       └── FFT + Hamming窗 + fftshift
  • 通道1 RDMAP：data1_rdmap
  • 通道2 RDMAP：data2_rdmap
  • 相干积累 RDMAP：data1_com + data2_com 后做多普勒 FFT

4. 目标检测 (第684-1372行)

三级检测流水线：

4.1 5×5 区域分割 (perform5x5RegionDivision, 第1016行)

  • 将 RDMAP 按距离维5等分、多普勒维5等分
  • 每区域设置独立 CFAR 阈值（中心零速区设阈值50=不检测）

4.2 3×3 邻域峰值检测 (performPeakDetection3x3, 第916行)

  • 动态阈值：同距离单元所有速度维功率均值 × 100
  • 3×3 邻域局部最大值判定

4.3 CA-CFAR 验证 (performCACFAROnPeaksWithRegionThresholds, 第726行)

  • 十字参考窗：距离维 + 多普勒维（共32个参考单元）
  • 保护单元：距离2，多普勒2
  • 边界处理：速度维循环（wrap-around），距离维镜像复制
  • 修剪均值（trimmedMean）：去4大0小

5. 双通道测角 (第1379-1607行)

calculateSimpleTargetInfo() 函数：

  1. 抛物线插值：距离维和速度维分别进行3点抛物线插值提升精度
  2. 干涉仪测角： 
    Δφ = angle(data2) - angle(data1)
sin(θ) = λ·Δφ / (2π·d)     (d = 68mm 天线间距)
azimuth = arcsin(sin(θ))
  3. 目标聚类：去除运动假目标
    • 距离差 < 9m
    • 速度差 < 0.5m/s
    • 方位角差 < 7°
    • 聚类组中保留功率最大的主目标，其余标记为聚类目标(不参与跟踪)

6. EKF 多目标跟踪 (第1816-2477行)

6.1 状态模型

  • 状态向量：6维 [x, vx, y, vy, z, vz]
  • 运动模型：CV（匀速）模型
  • 过程噪声 Q：diag([0.1, 0.01, 0.1, 0.01, 0.1, 0.01])

6.2 测量模型

  • 测量向量：4维 [range, velocity, azimuth, elevation]
  • 非线性测量方程（直角→极坐标）
  • 雅可比矩阵 H：解析计算
  • 测量噪声 R：diag([0.2², 1², (6°)², 0²])

6.3 数据关联

  • 方法：最临近匹配（Nearest Neighbor）
  • 代价函数：加权综合代价 = 0.6×距离差 + 1.0×速度差 + 0.5×方位角差 + 0.4×俯仰角差
  • 关联门限：距离9m, 速度2m/s, 方位角10°, 俯仰角0.1°
  • 新目标代价×3（避免误匹配）

6.4 生命周期管理

生命值规则：
  ┌────────────────────────────────────────────┐
  │ 第1帧检测 → 新跟踪 (life=1)                │
  │ 匹配成功    → life+1 (上限 max_life=100)    │
  │ 匹配失败    → life-1                        │
  │ life ≤ 0   → 目标消亡，序号回收             │
  │ life ≤ 2   → 隐藏（不显示）                 │
  │ life > 2   → 显示                           │
  └────────────────────────────────────────────┘

6.5 序号重用机制

  • 消亡目标的序号回收到 available_ids 池
  • 新建跟踪时优先使用回收序号
  • 可用池上限50个

7. 可视化显示 (第1873-2829行)

7.1 雷达扇形视图 (左侧)

  • 扇形区域：±10°, 距离 100-1100m
  • 距离环：391.68m, 783.36m
  • 跟踪目标：按速度用颜色区分的序号文本
    • 黑色：-2 ~ +2 m/s（低速/静止）
    • 蓝色：≤ -2 m/s（靠近）
    • 绿色：≥ +2 m/s（远离）
  • 检测目标：透明圆点（可开关，默认显示）
    • 青色：聚类目标
    • 其他颜色同跟踪目标规则

7.2 目标列表 (右侧)

UITable 显示：序号 | 距离(m) | 速度(m/s) | 方位角(°) | 俯仰角(°) | 强度(dB) | 信噪比 | 生命值

运行方式

  1. 在 MATLAB 中打开 AMEKFASSO20251218.m
  2. 确保所有 .m 文件在同一目录或 MATLAB 路径中
  3. 运行主函数，在弹出的文件选择对话框中选择 .dat 回波数据文件
  4. 程序将循环处理数据并实时显示跟踪结果
